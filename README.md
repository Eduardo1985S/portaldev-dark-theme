## ✨ Sobre

O **PortalDev Dark** foi criado para desenvolvedores que passam longas horas programando e precisam de um ambiente visual confortável, com contraste equilibrado e cores cuidadosamente escolhidas para reduzir o cansaço visual.

## 🎨 Paleta de Cores

| Elemento | Cor | Hex |
|---|---|---|
| Fundo do Editor | 🟫 | `#1B1E2B` |
| Texto Principal | ⬜ | `#E8E8E8` |
| Palavras-chave | 🔵 | `#4FC1FF` |
| Strings | 🟢 | `#4EC9B0` |
| Avisos | 🟡 | `#E2B93D` |
| Erros | 🔴 | `#F44747` |
| Brackets | 🟡🟣🔵 | Rainbow |

## 🚀 Instalação

1. Abra o **Visual Studio Code**
2. Vá em **Extensões** (`Ctrl+Shift+X`)
3. Pesquise por **PortalDev Dark Theme**
4. Clique em **Instalar**
5. Vá em **Preferências → Tema de Cores** e selecione **PortalDev Dark**

> Ou instale diretamente pelo terminal:
> ```
> code --install-extension portaldev.portaldev-dark-theme
> ```

## 🛠️ Recursos

- **Semantic Highlighting** — destaque inteligente baseado na semântica do código
- **Brackets coloridos** — identificação visual rápida de pares de chaves/colchetes
- **Contraste equilibrado** — pensado para longas sessões de código
- **UI integrada** — sidebar, terminal, abas, painéis e menus com cores harmonizadas

## 📦 Ativação Manual

Se preferir configurar manualmente, adicione ao seu `settings.json`:

```json
{
  "workbench.colorTheme": "PortalDev Dark"
}
```
